# Jiji Website

Documentation and landing page for [Jiji](https://github.com/acidtib/jiji), the open-source container orchestration tool.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build

```bash
npm run build
```

Outputs static site to `./out` directory.

## Stack

- [Next.js 16](https://nextjs.org/) - React framework
- [Nextra v4](https://nextra.site/) - Documentation theme
- [Tailwind CSS](https://tailwindcss.com/) - Styling
- [shadcn/ui](https://ui.shadcn.com/) - UI components

## Deployment

Automatically deployed to GitHub Pages on push to `main` via GitHub Actions.

## Related
